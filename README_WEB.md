# 자리 배치 프로그램 - 웹버전

## 실행 방법

1. 터미널 열기
2. `python web_server.py` 입력
3. 브라우저에서 `http://localhost:8000` 접속

## 기능

- 자리 배치: 그룹 제약조건 고려하여 자동 배치
- 배치 기록: 배치 번호와 시간 기록 (최대 10개)
- 상태 유지: 새로고침시 배치 결과 유지, 서버 재시작시 기록 초기화  
- 드래그 앤 드롭: 개별 자리 및 모둠 전체 교환 가능
- PNG 저장: 배치도를 이미지 파일로 다운로드
- 설정: 멤버/그룹 편집 가능
